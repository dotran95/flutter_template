# App

A new Flutter project.

# Setup

FLUTTER_VERSION: 3.22.0
XCODE_VERSION: 15.2
COCOAPOD_VERSIONL: 1.15.2

###### Install [FVM](https://fvm.app/docs/getting_started/installation)

```sh
brew tap leoafarias/fvm
brew install fvm
```

###### Install [Yq](https://github.com/mikefarah/yq/#install)

```sh
brew install yq
```


###### Install Flutter SDK

```sh
make setup
```
# Run

##### Dev Environment

```sh
make dev
```

##### Stg Environment

```sh
make stg
```

##### Prod Environment

```sh
make prod
```

# Build
##### Dev Environment
```sh
make build/ios/dev
```

##### Stg Environment
```sh
make build/ios/stg
```
